# Raspberry Pi Helpdesk Labs

This repository documents hands-on IT helpdesk labs performed using a Raspberry Pi in a home lab environment.

## Overview
These labs simulate real-world IT support scenarios such as network issues, DNS failures, and system troubleshooting. Each lab follows a structured approach: identify the issue, diagnose the root cause, and apply a solution.

## Completed Labs

### DNS Troubleshooting Lab
Simulated a network issue where a user could not access the internet due to DNS misconfiguration.

View full lab:  
[dns-troubleshooting.md](dns-troubleshooting.md)

### Key Skills Demonstrated
- Network troubleshooting  
- DNS configuration and analysis  
- Linux command line usage  
- Remote system access via SSH  
- Problem isolation and resolution  

## Tools & Technologies
- Raspberry Pi (Raspberry Pi OS)  
- SSH (remote access from Windows)  
- Linux CLI tools: `ping`, `nslookup`, `dig`
